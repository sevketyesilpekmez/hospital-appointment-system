# Hospital Appointment System

A Java OOP-based hospital appointment management system.

## Features
- Add doctors
- Add patients
- Create appointments
- Display appointment list
- OOP design with inheritance and interfaces

## Technologies
- Java
- Object-Oriented Programming
- Collections (ArrayList)
- Exception Handling

## Project Structure
- Doctor
- Patient
- Appointment
- HospitalService
- Order system (Lab, Medicare, External, etc.)

## How to Run
1. Open the project in an IDE (IntelliJ / Eclipse)
2. Run Main.java
